Family Tree Creator

Instructions:

1. Start by creating the first family member

2. Select any member to add relatives (parent, child, sibling, spouse)

3. Click on members to view or edit their information

4. Use the zoom controls to navigate larger family trees

Keyboard Controls:

• Arrow keys: Navigate the tree

• +/- keys: Zoom in/out

• Enter: Select highlighted member

• Esc: Cancel current action
